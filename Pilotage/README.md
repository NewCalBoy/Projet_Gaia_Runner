# Partie 3 : Pilotage
## Description
Cette sous partie concerne la commande du Gaïa Runner, c'est-à-dire l'usage du module Bluetooth pour controler le robot à distance.

## Solution technique
Pour le pilotage du robot, la solution choisit est d'utiliser un module Bluetooth HC05. Il sera connecté sur le PCB principal.
